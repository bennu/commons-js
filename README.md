## üíª Usage

### JavaScript (CommonJS)

```javascript

// Basic validation

```

### TypeScript / ES Modules

```typescript
import { isValidRut} from 'common-js';

// Basic validation
console.log(isValidRut('12.345.678-5')); // true
```
